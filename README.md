Este desafío forma parte del programa **Oracle Next Education** en colaboración con **Alura Latam**.

---

# 📌 Descripción del Proyecto

Este proyecto realiza un **Análisis Exploratorio de Datos (EDA)** para identificar patrones y factores asociados con la cancelación de clientes en Telecom X.  
Utilizando Python y diversas bibliotecas de análisis de datos, se recopila información desde una **API**, se procesa mediante un flujo **ETL (Extracción, Transformación y Carga)** y se generan visualizaciones que permiten identificar tendencias y comportamientos de los clientes.

---

# ❓ Problema que resuelve

Telecom X enfrenta una **alta tasa de cancelación de clientes**, lo cual impacta negativamente en sus ingresos y crecimiento.

Este análisis busca identificar los **factores principales que influyen en la pérdida de clientes**, permitiendo que el equipo de **Data Science** desarrolle modelos predictivos y estrategias que ayuden a **mejorar la retención de clientes**.

# 🔄 Proceso de Análisis (ETL)

El proyecto sigue un flujo de trabajo basado en **ETL (Extracción, Transformación y Carga)**.

### 1️⃣ Extracción
Obtención de datos desde una **API**, lo que permite acceder a la información de los clientes de manera estructurada.

### 2️⃣ Transformación
En esta etapa se preparan los datos para el análisis mediante:

- Limpieza de datos  
- Eliminación o tratamiento de valores nulos  
- Conversión de tipos de datos  
- Estandarización de variables  

### 3️⃣ Análisis
Se realiza un **Análisis Exploratorio de Datos (EDA)** que incluye:

- Exploración de variables  
- Identificación de patrones y tendencias  
- Creación de visualizaciones para interpretar los resultados

# ▶️ Uso del Proyecto

El proyecto permite analizar distintos factores relacionados con la evasión de clientes, como:

✔️ Distribución de clientes que cancelan el servicio
✔️ Relación entre tipo de contrato y churn
✔️ Impacto del costo mensual en la cancelación
✔️ Comportamiento de clientes según servicios contratados
✔️ Identificación de patrones que influyen en la pérdida de clientes

Para facilitar la interpretación de los datos se generan diferentes visualizaciones, como:

- 📊 Gráficos de barras

- 📈 Gráficos de correlación

- 🥧 Gráficos de distribución

- 🔥 Mapas de calor (heatmaps)

Estas visualizaciones permiten comprender mejor los factores que influyen en la cancelación de clientes.
